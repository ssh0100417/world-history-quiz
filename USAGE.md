# Netlify 배포 후 사용법

## 모바일 접속

1. Netlify 배포 URL을 모바일 브라우저에서 엽니다.
2. iPhone Safari: 공유 버튼을 누른 뒤 `홈 화면에 추가`를 선택합니다.
3. Android Chrome: 메뉴 버튼을 누른 뒤 `홈 화면에 추가` 또는 `앱 설치`를 선택합니다.

## 문제 풀이

1. 첫 화면에서 버전과 주차를 선택합니다.
2. 문항 수를 선택합니다.
3. 테스트를 시작합니다.
4. 오답은 오답 노트 화면에서 확인합니다.

## 배포 업데이트

문항을 수정한 뒤 GitHub 저장소에 변경사항을 올리면 Netlify가 자동으로 새 배포를 만듭니다.
